# 第一章：用演算法思維整理日常工作

::: advisory
本章目標
:::

- 建立演算法思維：把日常工作轉成輸入、輸出、限制、成本
- 透過可執行範例與測試，養成可重現的工作流程

::: advisory
情境與限制
:::

- 情境：你必須在有限時間內整理待辦事項並排出優先順序
- 輸入：任務清單、優先度、預估時間、截止日
- 輸出：排序後的行事曆與工作建議
- 限制：必須保留可測試與可重現步驟

::: advisory
可重現規則
:::

- 本章程式碼位置： `examples/ch01/`
- 本章核心模組： `src/algo_in_life/`
- 測試： `pytest -q tests/test_ch01_time_log.py`
- 資料： `data/synthetic/`

## 本章內容概要

本章將示範如何把「每天要做的事情」轉成程式可以理解的問題，並用簡單的排序與時間估算建立可執行的工作流程。

請參考 `examples/ch01/time_log.py`。
